# AeroVision AI Frontend

## Overview
This is the frontend interface for AeroVision AI, providing a modern React-based drone control dashboard.

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Features
- React 18 with TypeScript
- Vite build system
- Tailwind CSS for styling
- PWA support
- Real-time drone telemetry
- Interactive map display
- AI-powered command interface
- Fleet management dashboard
- Responsive design

## Components
- **AppSimpleChat.tsx** - Main application component
- **services/DroneService.ts** - Drone communication service
- **components/** - Reusable UI components
- **assets/** - Static assets and images

## Configuration
- **vite.config.ts** - Vite build configuration
- **tailwind.config.js** - Tailwind CSS configuration
- **tsconfig.json** - TypeScript configuration

## Environment
- Development server runs on port 5173
- Backend API should be available on port 3001
- PWA features enabled for mobile deployment

## Dependencies
- React 18 for UI framework
- Vite for build tooling
- Tailwind CSS for styling
- TypeScript for type safety
- PWA plugin for mobile support
